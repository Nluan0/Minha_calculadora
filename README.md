**1. Clone o repositório**
```bash
git clone https://github.com/Nluan0/Minha_calculadora.git
```

**2. Entre na pasta**
```bash
cd Minha_calculadora
```

**3. Execute o projeto**
```bash
python main.py
```

> Não é necessário instalar nenhuma dependência externa, tudo utilizado é nativo do Python.
