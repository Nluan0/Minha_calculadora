Calculadora em Python 🧮
Descrição

A Calculadora em Python é uma aplicação desktop desenvolvida com Tkinter, criada para realizar operações matemáticas básicas de forma simples, prática e visual. O projeto permite ao usuário efetuar cálculos de adição, subtração, multiplicação e divisão por meio de uma interface gráfica intuitiva.

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais da programação, como funções, estruturas condicionais, manipulação de variáveis e criação de interfaces gráficas com Python.

Tecnologias utilizadas

Python — lógica e funcionamento da aplicação

Tkinter — interface gráfica da calculadora

Como funciona

O usuário clica nos botões numéricos para inserir os valores.

Em seguida, seleciona a operação desejada.

Depois, informa o segundo valor.

Ao clicar em =, o sistema realiza o cálculo e exibe o resultado na tela.

O botão C limpa os dados para iniciar uma nova operação.

Objetivo do projeto

Este projeto foi criado para fortalecer meus conhecimentos em Python e no desenvolvimento de aplicações com interface gráfica, colocando em prática a lógica de programação de forma visual e interativa.
